
# SkillMetric: Python Proficiency Assessment Tool

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue.svg)
![GUI](https://img.shields.io/badge/UI-Tkinter-orange.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

**SkillMetric** is a lightweight desktop application designed to evaluate Python programming skills through an interactive testing environment. It assesses candidates across three core domains: Syntax, Algorithms, and Object-Oriented Programming (OOP).

---

## 🚀 Key Features

*   **Comprehensive Question Bank:** 50 curated questions with varying difficulty levels (1–3 points).
*   **Dynamic Ranking System:** Automatically categorizes performance from **Junior** to **Master**.
*   **Real-time Analytics:** Tracks performance by category to identify the candidate's "Top Skill."
*   **Global Leaderboard:** Local storage (JSON) to track and display the top 10 highest-scoring candidates.
*   **Modern UI:** Clean, distraction-free interface built with Python's Tkinter.

---

## 🛠 Tech Stack

- **Language:** Python 3.10+
- **Framework:** Tkinter (Built-in GUI)
- **Data Persistence:** JSON
- **OS:** Cross-platform (Windows, macOS, Linux)
